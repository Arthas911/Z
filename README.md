3. 清风舞明月，幽梦落花间。一梦醒来，恍如隔世，两眉间，相思尽染。只身天涯，独醉贪欢。

4. 一场红尘恋，一份千年缘，几缕隔岸相思，隐逸了多少楼台旧梦？在水一方，愿着一身荷香，
